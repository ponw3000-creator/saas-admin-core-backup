# 跨端自适应设计规范 (Responsive Design Rules)

## 1. 核心断点标准

严格采用 Element Plus 体系，所有页面必须遵守：

| 断点 | 设备类型 | 宽度范围 |
|------|----------|----------|
| xs | 移动端 | < 768px |
| sm | 平板 | ≥ 768px |
| md | 小桌面 | ≥ 992px |
| lg | 标准桌面 | ≥ 1200px |

## 2. 布局降级原则

### 桌面端（md 及以上）
- 允许使用多栏 Flex/Grid 布局
- 可使用左右分栏、卡片栅格等复杂布局
- 侧边栏、装饰性元素可正常展示

### 移动端（xs 和 sm）
- **强制转为单列上下堆叠布局**（flex-direction: column）
- **隐藏非核心视觉装饰元素**（品牌展示区、装饰性图形、侧边栏）
- **表单容器宽度必须设为 100%**
- **保留适度内边距**（如 padding: 20px）
- 按钮和输入框需占满可用宽度

## 3. 响应式实现规范

### CSS 媒体查询使用

```scss
/* 移动端优先：默认单列布局 */
.container {
  padding: 20px;
  width: 100%;
}

/* 平板及以上（sm, md, lg） */
@media (min-width: 768px) {
  .container {
    padding: 40px;
  }
}

/* 桌面端（md 及以上）允许多栏布局 */
@media (min-width: 992px) {
  .container {
    display: flex;
    /* 多栏布局逻辑 */
  }
}
```

### Element Plus 响应式辅助类

- `hidden-xs-only` / `hidden-sm-and-up` 等辅助类控制元素显示隐藏
- 布局容器使用 `el-container`、`el-aside`、`el-main`

## 4. 表单响应式规范

- 移动端表单标签位置推荐 `label-position="top"`
- 输入框宽度 100%，配合适度内边距
- 按钮宽度在移动端应占满容器（width: 100%）

## 5. 登录页专项规范

### 桌面端（md 及以上）
- 采用左右双栏布局
- 左侧占 50%（品牌展示、Slogan，主色调背景）
- 右侧占 50%（纯白背景居中表单，最大宽度 400px）

### 移动端（sm 及以下）
- 自动隐藏左侧品牌展示区
- 右侧表单区占满屏幕 100%
- 表单居中显示，表单元素占满可用宽度
